# Business Glossary

Domain terms for Bap Number Adventure.

## Glossary

| Term | Definition |
|---|---|
| **Bắp** | Mascot character — a friendly corn cob that guides the child |
| **World** | A themed game world on the world map (e.g., Farm, Space, Ocean) |
| **Level** | A single game session within a world |
| **Mini-Game** | One of 5 game types: Hear & Tap, Number Order, Build Number, Even/Odd, Math Kitchen |
| **Hear & Tap** | Listen to a spoken number, tap the correct tile |
| **Number Order** | Find the missing number in a sequence |
| **Build Number** | Drag digits to construct a target number |
| **Even/Odd** | Classify numbers as even or odd |
| **Math Kitchen** | Simple addition/subtraction in a cooking theme |
| **Sticker** | Reward item earned by completing levels |
| **Streak** | Consecutive daily play sessions |
| **Parent Gate** | PIN/pattern screen to access parent area |
| **Difficulty** | easy / medium / hard — auto-adjusted based on performance |
| **Star Rating** | 1–3 stars earned per level based on accuracy & speed |
| **Progress** | Persistent state tracking completed levels, stars, stickers |
