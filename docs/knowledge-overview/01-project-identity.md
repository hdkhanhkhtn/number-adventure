# Project Identity

## Name
**Bap Number Adventure** (Bắp Number Adventure)

## Tagline
Learning numbers, one adventure at a time.

## Vision
A playful, warmly-designed educational web app that makes early math thinking exciting for preschool children (age 3–7) and keeps parents informed and confident.

## Purpose
Help young children gradually build strong number sense through gamified mini-games, emotional encouragement, and a magical world adventure. Designed for mobile-first interaction with audio-first guidance, minimal text, and progressive difficulty.

## Problem Statement
Young children struggle to engage with traditional number-learning methods. Screen time is inevitable — this app makes it productive and joyful while giving parents visibility and control.

## Target Learning Goals
- **Number fluency:** Recognize and work with tens, hundreds, thousands
- **Number classification:** Understand even and odd numbers
- **Basic arithmetic:** Introduction to addition and subtraction (within 10)
- **Number sense:** Strengthen intuitive understanding through playful repetition and visual learning

## Core Brand
**Mascot:** Bắp — a warm, supportive character companion who guides, celebrates, and encourages
**Theme:** Magical learning worlds: Number Forest, Tens Town, Big Number Castle, Even/Odd Bridge, Math Kitchen
**Style:** Warm, playful, safe, colorful; cute but not cluttered; rounded shapes; rewarding animations

## Target Users

| User | Role | Learning Goals | Needs |
|---|---|---|---|
| **Child (4 yrs)** | Primary | Number recognition, tens/ones, early arithmetic | Audio guidance, large touch targets, immediate reward, short sessions |
| **Parent** | Secondary | Track progress, ensure safety, manage screen time | Dashboard overview, progress trends, settings control |

## MVP Scope
**Child Features:**
- Home dashboard with world map and daily goals
- 5 mini-games (Hear & Tap, Number Order, Build the Number, Even/Odd, Math Kitchen)
- Reward system with celebration screens and stickers
- Daily streak tracking

**Parent Features:**
- Parent gate (PIN protection)
- Dashboard home with overview of child progress
- Progress details with trend analysis
- Settings for audio, language, and session time limits

**Delivery:** Next.js 14+ web app with TypeScript, TailwindCSS, Framer Motion, mobile-first responsive design
