# Microcopy Guide

## Tone & Voice

**For Child-Facing Screens:**
- Warm, encouraging, never punishing
- Short sentences (≤5 words)
- Positive, celebratory language
- Action-oriented (what to do, not what NOT to do)
- Playful but clear
- No academic tone
- Use simple, conversational English

**For Parent Dashboard:**
- Clear, informative, calm
- Professional but warm
- Helpful (explain what's happening)
- Data-focused (numbers, trends)
- Empowering (give parents control)

**Golden Rules:**
- Always encouraging, never shaming
- Short > long
- Show > tell (prefer icons + visuals)
- No jargon
- No pressure

## Guidelines for New Copy

1. **Keep it short** — max 5-7 words per line
2. **Use imperative** — "Tap the number" not "Please tap the number"
3. **Celebrate effort** — "Great job!" not "That's correct"
4. **Stay positive** — "Try again" not "Wrong, try again"
5. **Provide guidance** — "Listen carefully" not "Pay attention"
6. **Use audio first** — Let audio do the heavy lifting; visual supports it
7. **Bilingual friendly** — Ensure English and Vietnamese both fit the space
8. **No technical jargon** — Replace "submit" with action verbs: "Check", "Done", "Go"
9. **Test readability** — Ensure text is large enough (min 16px for child screens)
10. **Use icons when possible** — Reduce text dependency with visual symbols

## Language Options

The app supports bilingual mode. Ensure all copy can be toggled between:
- **English only** — all UI in English
- **Vietnamese only** — all UI in Vietnamese
- **Bilingual** — child prompts in English (audio), labels in Vietnamese
