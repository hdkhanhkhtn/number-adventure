# PROMPT – Parent Progress Details

```markdown
Design the detailed progress tracking screens for parents in “Bap Number Adventure”.

## Purpose
Parents should be able to understand what the child is learning, where they are doing well, and where they need more support.

## Required Progress Sections
1. Overall learning progress
2. Skill breakdown
3. Recent sessions
4. Areas needing review
5. Milestones achieved

## Skill Categories
- number recognition
- listening and matching
- counting sequence
- tens and ones
- hundreds awareness
- even / odd
- basic addition
- basic subtraction

## Data Visualization Style
Use simple, mobile-friendly visualizations such as:
- progress bars
- skill cards
- streak indicators
- timeline list
- simple charts if needed

Avoid overly complex analytics.

## Additional Features
- suggested next activities
- replay recommended games
- confidence or mastery indicator
- date filters if useful

## Output
Design polished mobile-first screens for parent progress reporting.
```
