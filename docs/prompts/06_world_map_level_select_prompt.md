# PROMPT – World Map / Level Select

```markdown
Design the World Map and Level Selection screens for “Bap Number Adventure”.

## Purpose
The child progresses through playful math worlds. The map should show a sense of adventure and progression while staying extremely easy to understand.

## Worlds in MVP
- Number Forest
- Tens Town
- Big Number Castle
- Even/Odd Bridge
- Math Kitchen

## Required Screens
1. World map overview
2. Single world detail screen
3. Level node / lesson selection screen
4. Locked world state
5. Completed world state

## World Map Requirements
- whimsical illustrated map
- path-based progression
- each world represented by a distinct landmark
- progress shown visually through stars / path nodes / completion rings
- current world highlighted
- future worlds visible but possibly locked
- simple tap interaction

## Single World Screen Requirements
- world title
- short friendly description
- mascot encouragement
- list or node path of lessons
- lesson status:
  - locked
  - available
  - in progress
  - completed
- CTA to continue from last lesson

## Lesson Tile / Node Requirements
- lesson number or icon
- mini objective
- reward preview
- completion state
- difficulty or skill indicator if needed

## UX Notes
- the map must feel rewarding and exploratory
- it should not feel like homework
- use strong visual metaphors and very clear states

## Output
Design polished mobile-first screens and define how the world map scales to tablet.
```
