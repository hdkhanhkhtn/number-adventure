## Gợi ý thêm cuối prompt

### Nếu muốn thiên về Figma-ready

```markdown
Please organize the output as a clear design system and screen set suitable for Figma production workflow, with reusable components and consistent spacing rules.
```

### Nếu muốn thiên về app hiện đại

```markdown
Use a polished, modern product design style similar to high-quality educational apps, while remaining original, child-friendly, and brandable.
```

### Nếu muốn thiên về prototype clickable

```markdown
Prioritize realistic app flows and screen transitions so the result can be easily turned into a clickable prototype.
```

### Nếu muốn UI sạch, ít rối

```markdown
Keep the interface calm, uncluttered, and focused, avoiding over-decoration or excessive visual noise.
```
