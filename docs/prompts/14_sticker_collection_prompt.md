# PROMPT – Sticker / Collection Screen

```markdown
Design the collectible / sticker collection screen for “Bap Number Adventure”.

## Purpose
Children earn stickers, badges, or collectible items through gameplay. The collection screen should motivate continued engagement and visually represent progress.

## Requirements
- display owned stickers
- show locked stickers
- categorize by world or theme
- allow child to tap a sticker to see it larger
- show how to earn locked items
- simple and fun grid or album-style layout

## Content Suggestions
Collections may include:
- number animals
- world badges
- learning trophies
- character accessories
- magical objects

## UX Constraints
- very visual
- minimal text
- simple navigation
- clear reward loop

## States
- empty / just started
- partially filled collection
- unlocked new sticker highlighted
- full set completion celebration

## Output
Provide polished mobile-first screen designs.
```
