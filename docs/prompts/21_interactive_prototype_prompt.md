# PROMPT – Interactive Prototype

```markdown
Create a clickable prototype flow for the MVP of “Bap Number Adventure”.

## Goal
Show a realistic end-to-end mobile-first user journey for both child mode and parent mode.

## Prototype Flows to Include
### Child Flow
1. Open app
2. Enter child home
3. Tap Play Now
4. Open current world
5. Start Hear and Tap Number
6. Answer correctly
7. Continue to Missing Number
8. Finish short session
9. View reward screen
10. View updated progress / sticker unlock

### Parent Flow
1. Enter parent gate
2. Open parent dashboard
3. View progress details
4. Adjust language settings
5. Set daily time goal
6. Return to child home

## Interaction Notes
- simulate button transitions
- include microinteractions
- show success feedback
- show hint behavior
- show locked/unlocked state transitions

## Output
Provide a prototype-ready screen sequence and interaction logic.
```
