# PROMPT – Mini-game 3: Build the Number

```markdown
Design the UI for a mini-game called “Build the Number” in the app “Bap Number Adventure”.

## Learning Goal
Help the child understand number structure using tens and ones, and later hundreds.

## Core Concept
The child is shown a target number, such as 24, and must build it using:
- tens blocks / bundles
- ones blocks / units

Later the game can scale to:
- hundreds blocks
- thousands icons

## Interaction
- target number displayed clearly
- draggable tens and ones blocks
- drop zones or build area
- visual counting support
- submit/check button or auto-check

## Requirements
- highly visual
- Montessori-inspired but playful
- blocks should be easy to drag
- show the difference between tens and ones clearly
- use soft color coding
- keep the interface simple

## Variants to Design
1. Tens and ones only
2. Read the number and build it
3. Build based on audio prompt
4. Match built blocks to written number
5. Intro version with guided hints

## Show These UI States
- empty build area
- partially completed build
- correct construction
- incorrect construction
- hint overlay explaining “2 tens and 4 ones”

## Output
Design polished mobile screen UI and the core reusable components for number building.
```
