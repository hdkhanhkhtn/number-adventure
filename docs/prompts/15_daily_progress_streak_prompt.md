# PROMPT – Daily Progress / Streak

```markdown
Design a daily progress / streak screen for “Bap Number Adventure”.

## Purpose
Show the child and parent that learning is happening consistently. The screen should motivate continuation, not pressure.

## Requirements
- daily goal status
- stars earned today
- lessons completed today
- streak or consistency display
- simple celebration if daily goal is met
- motivational mascot message

## Audience
Primarily child-facing, but can be understandable to parents too.

## UX Style
- visual
- positive
- very simple
- warm and rewarding

## States
1. Before daily session
2. During progress
3. Goal completed
4. Missed day but encouraging recovery

## Output
Design polished mobile screens and reusable progress cards.
```
