# PROMPT – Illustration / Art Direction

```markdown
Create an illustration and art direction guide for “Bap Number Adventure”, a mobile-first educational app for preschool children.

## Product Mood
- magical
- cheerful
- cozy
- playful
- safe
- adventurous

## Audience
Children around age 4 and their parents.

## Illustration Goals
- support learning
- create emotional warmth
- make the app memorable
- help distinguish worlds and game types
- never distract excessively from tasks

## Create Direction For
- mascot character
- world backgrounds
- game props
- stickers / collectibles
- reward elements
- parent mode illustration tone

## World Themes
- Number Forest
- Tens Town
- Big Number Castle
- Even/Odd Bridge
- Math Kitchen

## Style Requirements
- simple shapes
- rounded forms
- soft outlines
- gentle facial expressions
- child-friendly color use
- scalable illustration system
- suitable for UI integration

## Output
Provide art direction guidelines and example illustration usage recommendations.
```
