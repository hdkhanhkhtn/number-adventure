# PROMPT – Mini-game 5: Basic Add / Take Away

```markdown
Design the UI for a mini-game called “Math Kitchen” or “Add and Take Away” in the app “Bap Number Adventure”.

## Learning Goal
Introduce basic addition and subtraction through visual and story-based interactions.

## Core Concept
The child sees objects being added or taken away, then answers the result.
Examples:
- 3 apples + 2 apples = ?
- 5 cookies - 1 cookie = ?

## Interaction Styles
1. Visual counting with answer choices
2. Drag more objects into a bowl/plate
3. Remove objects and count what remains
4. Story prompt with large illustrations

## UI Requirements
- large central illustration area
- very simple equation display
- clear quantity visuals
- optional audio prompt
- answer choices as big number buttons
- highly engaging but calm

## Theme
Use a cozy theme such as:
- kitchen
- picnic
- snack time
- feeding a character

## Difficulty Levels
1. Addition within 5
2. Subtraction within 5
3. Addition within 10
4. Take away with clear visual support

## States to Show
- default
- objects being added
- objects being removed
- correct answer
- incorrect answer
- hint / recount mode

## Output
Design polished mobile screens that make early arithmetic feel playful and intuitive.
```
