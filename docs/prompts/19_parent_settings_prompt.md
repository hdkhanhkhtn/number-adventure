# PROMPT – Parent Settings

```markdown
Design the Parent Settings screens for “Bap Number Adventure”.

## Purpose
Give parents control over the child’s learning environment, while keeping settings simple and easy to use.

## Required Settings
### Learning Settings
- language mode: English / Vietnamese / Bilingual
- audio on/off
- guided hints on/off
- difficulty adaptation on/off or adjustable
- start level

### Session Settings
- daily time limit
- daily goal length
- allow free play
- bedtime / restricted time suggestions (optional)

### Profile Settings
- child profile edit
- avatar
- age / stage
- reset progress (with confirmation)

### Reward Settings
- celebration animation intensity
- sticker rewards on/off (if desired)
- sound effects volume

## UX Style
- calm
- parent-friendly
- clean
- organized into sections
- clear toggles and cards

## Output
Design polished mobile screens for settings and key modals.
```
