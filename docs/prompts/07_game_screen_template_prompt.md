# PROMPT – Game Screen Template

```markdown
Create a reusable game screen template system for a mobile-first preschool math app called “Bap Number Adventure”.

## Purpose
This template system should support multiple mini-game formats while staying visually consistent.

## General UX Goals
- one task per screen
- clear visual focus
- minimal text
- audio-first prompts
- big interaction areas
- immediate feedback
- supportive retry states

## Common Elements Across All Game Screens
- top progress bar
- close/back button
- audio replay button
- mascot hint area
- main prompt area
- interactive answer area
- optional visual aid area
- positive feedback animation area

## Variants Needed
1. Multiple choice tap game
2. Drag and drop game
3. Sequence arrangement game
4. Build the number block game
5. Simple arithmetic visual game

## Feedback States
- correct answer
- incorrect answer
- hint state
- next question transition

## Child-Friendly Rules
- avoid too many answer options
- large numbers and clear typography
- soft illustration support
- concise prompts
- low distraction

## Output
Design a component-based system showing:
- shared layout structure
- reusable component patterns
- how each mini-game fits into the template
- examples on mobile
```
