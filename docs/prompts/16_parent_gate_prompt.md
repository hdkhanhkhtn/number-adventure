# PROMPT – Parent Gate

```markdown
Design a Parent Gate screen for a preschool learning app called “Bap Number Adventure”.

## Purpose
Protect parent-only areas from accidental child access while keeping the experience lightweight and friendly.

## Requirements
- a simple child-resistant gate
- not scary or overly formal
- can use a parent challenge such as:
  - hold button for 3 seconds
  - solve a simple adult prompt
  - tap hidden pattern
- should be easy for a parent, hard for a young child to pass accidentally

## Include
- friendly parent mode label
- simple instructions
- cancel / back option
- clean design consistent with the app

## Output
Design a polished mobile-first parent gate screen.
```
