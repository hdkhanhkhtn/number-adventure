# PROMPT – Child Home Screen

```markdown
Design the child home screen for “Bap Number Adventure”, a mobile-first early math learning app for preschool children.

## Purpose
The home screen should feel like the child’s safe and magical learning home. It should help the child quickly start playing and show progress in a motivating way.

## Primary Goals of the Screen
- Start today’s learning session quickly
- Show where the child is in their adventure
- Surface rewards and collectibles
- Make progress visible and exciting
- Keep navigation simple

## Required Elements
- greeting with child name
- mascot companion
- “Play Now” primary CTA
- daily goal card
- world map shortcut
- current learning world / last lesson
- stars earned
- sticker / collection shortcut
- simple streak or consistency indicator
- parent mode entry (hidden/subtle)

## Content Direction
The child should feel:
- welcomed
- excited
- proud
- curious

## Layout Suggestions
Top:
- greeting
- avatar
- mascot bubble

Middle:
- large play now card
- daily goal / progress card
- world map / current adventure card

Bottom:
- rewards
- stickers
- maybe a bottom nav with 3–4 simple icons

## Important Constraints
- very large buttons
- no dense text
- child-friendly icons
- progress shown visually, not text-heavy
- use cards and playful illustrations
- very few choices on screen to avoid overwhelm

## Include These Sections
1. Home state for returning child
2. Home state after completing session
3. Home state with new unlocked sticker
4. Home state when a new world is unlocked

## Output
Provide polished mobile screen designs and clear component reuse suggestions.
```
