# PROMPT – Onboarding Flow

```markdown
Design the onboarding flow for a mobile-first preschool math app called “Bap Number Adventure”.

## Purpose
The onboarding should help a parent quickly set up the app and create a child-friendly experience. The child may also be present during onboarding, so it should be playful and welcoming.

## Screens Needed
1. Splash screen
2. Welcome / intro screen
3. Choose language mode
4. Create child profile
5. Select age / learning stage
6. Choose mascot or theme
7. Parent preferences setup
8. Short readiness / assessment intro
9. Completion / enter app

## UX Requirements
- visually inviting
- simple and warm
- minimal friction
- easy for parent
- magical for child
- progress indicator visible
- large buttons
- clear next/back steps

## Specific Content to Include
### Splash screen
- app logo
- mascot
- soft animated feel
- short loading phrase

### Welcome screen
- brief explanation:
  “A playful way for your child to explore numbers and early math.”
- CTA: Get Started

### Language setup
- English only
- Vietnamese only
- Bilingual
- audio-first explanation

### Child profile setup
- child name
- age
- avatar selection
- optional cute icon or profile character

### Learning stage setup
Allow parent to choose starting level:
- already recognizes 1–20
- recognizes 1–100
- ready for tens/hundreds
- ready for early addition

### Mascot selection
Choose 1 mascot companion:
- bear
- bunny
- robot
- cat
Each option should have a cute card and preview.

### Parent preferences
- daily time goal
- allow sound by default
- show guided hints
- enable celebration animations

### Assessment intro
A short message explaining:
“The app will gently adapt to your child’s level through play.”

### Completion screen
- confirmation message
- mascot greeting child by name
- CTA: Start Adventure

## Design Notes
- all screens mobile-first
- polished MVP quality
- suitable for production
- include microcopy
- include visual guidance for progress bar and step indicators
```
