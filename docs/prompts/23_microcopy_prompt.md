# PROMPT – Microcopy cho toàn app

```markdown
Write microcopy for the mobile-first preschool math learning app “Bap Number Adventure”.

## Goals
The copy should feel:
- warm
- short
- encouraging
- child-friendly in child mode
- clear and helpful in parent mode

## Create Copy For
### Child Mode
- welcome messages
- play now
- try again
- great job
- you did it
- let’s go
- find the number
- listen carefully
- build the number
- even or odd?
- add more
- take away
- reward messages
- unlocked messages
- daily goal complete

### Parent Mode
- onboarding descriptions
- settings labels
- progress labels
- skill descriptions
- session settings
- recommendation labels
- confirmation modals

## Language
Create the copy in:
1. English
2. Vietnamese
Optionally include bilingual examples.

## Tone rules
- short sentences
- no academic tone
- no pressure
- no negative phrasing
- always encouraging

## Output
Group the copy by screen and by feature.
```
