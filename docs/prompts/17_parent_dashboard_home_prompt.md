# PROMPT – Parent Dashboard Home

```markdown
Design the Parent Dashboard home screen for “Bap Number Adventure”.

## Purpose
Parents need a simple overview of the child’s learning progress, current level, and app settings.

## Tone
- calm
- informative
- organized
- trustworthy
- warm

## Required Sections
- child profile summary
- current learning stage
- progress overview
- recent activity
- strengths / skills practiced
- recommended next focus
- time spent
- shortcut to settings

## Information to Show
- current world
- lessons completed
- accuracy trends
- favorite game types
- number topics learned:
  - number recognition
  - tens and ones
  - even/odd
  - addition/subtraction intro

## Layout Requirements
- mobile-first
- clean cards
- easy scanning
- parent-friendly
- fewer playful distractions than child mode, but still brand-consistent

## Output
Design a polished mobile parent dashboard home screen.
```
