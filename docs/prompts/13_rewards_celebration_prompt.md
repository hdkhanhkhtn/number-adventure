# PROMPT – Rewards / Celebration

```markdown
Design the reward and celebration screens for “Bap Number Adventure”.

## Purpose
Reward screens should reinforce motivation, celebrate learning, and encourage the child to continue. They should feel exciting but not overstimulating.

## Reward Moments to Design
1. Correct answer feedback
2. End of mini-game
3. End of daily session
4. New sticker unlocked
5. New world unlocked
6. Achievement / milestone earned

## Required Elements
- stars earned
- mascot celebration
- visual reward item
- encouraging message
- CTA to continue
- optional progress indication toward next unlock

## Tone
- joyful
- proud
- encouraging
- magical

## Variants Needed
- small reward popup
- full-screen celebration
- end-of-session summary
- collectible unlock screen

## Output
Design a family of reward UI screens and components with consistent visual language.
```
