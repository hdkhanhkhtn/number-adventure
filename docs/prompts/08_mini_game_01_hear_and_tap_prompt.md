# PROMPT – Mini-game 1: Hear and Tap Number

```markdown
Design the UI for a mini-game called “Hear and Tap Number” in the app “Bap Number Adventure”.

## Learning Goal
The child hears a number in English and taps the correct number on screen.

## Core Interaction
- audio prompt plays automatically or can be replayed
- 2 to 4 answer choices appear as large number buttons
- child taps one answer
- immediate feedback appears

## Requirements
- very large number cards
- audio replay button must be prominent
- optional mascot speech bubble:
  “Can you find thirty-two?”
- playful but clear interface
- no clutter

## Difficulty Variants
Design examples for:
1. Easy: choose between 2 options
2. Medium: choose between 3 options
3. Harder: choose between 4 options with visually similar numbers (e.g. 32, 23, 30, 42)

## States to Show
- default state
- after correct answer
- after incorrect answer
- hint state
- next question transition

## Visual Style
- bright and clean
- child-friendly
- number cards feel tappable and satisfying
- success animation with stars/confetti/light sparkle

## Output
Provide a polished mobile screen UI and state variants.
```
