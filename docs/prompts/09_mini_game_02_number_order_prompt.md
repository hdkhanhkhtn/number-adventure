# PROMPT – Mini-game 2: Number Order / Missing Number

```markdown
Design the UI for a mini-game called “Number Order” / “Missing Number” in the app “Bap Number Adventure”.

## Learning Goal
Help the child understand counting sequence, number order, and missing numbers.

## Game Types
### Type A: Missing Number
Example:
- 18, 19, __, 21
- 10, 20, __, 40
- 100, 200, __, 400

### Type B: Put in Order
The child drags or taps numbers into the correct order.

## Interaction Types
- tap to choose missing number
- drag card into empty slot
- reorder cards in a sequence

## Requirements
- very clear sequence slots
- number cards large and colorful
- clear empty state slot
- optional visual support line or train metaphor
- mascot guidance

## Theme Suggestions
Use a playful theme such as:
- number train
- stepping stones
- rainbow path
- bridge tiles

## Show These Variants
1. Simple 1-step forward sequence
2. Counting by tens
3. Counting by hundreds
4. Missing final number
5. Missing middle number

## Feedback States
- correct fill
- wrong attempt
- guided hint
- complete sequence celebration

## Output
Design polished mobile-first screens for both “missing number” and “put in order” variants.
```
