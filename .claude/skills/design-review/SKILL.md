# SKILL: Design Review

## Trigger
`design review` · `review ui` · `review figma`

## Checklist
1. **Consistency** — theo design system Workshopman?
2. **Mobile-first** — layout hoạt động ≤375px?
3. **Accessibility** — contrast ≥4.5:1, touch target ≥44px
4. **States** — Loading · Empty · Error · Disabled đã có?
5. **RBAC visual** — UI phản ánh đúng quyền của từng role?
6. **Domain accuracy** — thuật ngữ UI đúng với domain garage?
7. **Dev handoff** — spacing rõ? Assets export được?
